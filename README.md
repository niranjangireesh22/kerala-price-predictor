## рҹ§  Pipeline Overview

1. Load the dataset and drop non-predictive columns (`City/Place`, `date`).
2. Impute missing numeric values with the column median.
3. Detect and treat outliers using the IQR method (winsorization).
4. Encode categorical columns with `LabelEncoder`.
5. Scale features with `StandardScaler`.
6. Train a fully-connected neural network:
   `Dense(256) вҶ’ BatchNorm вҶ’ Dropout(0.3) вҶ’ Dense(128) вҶ’ Dropout(0.3) вҶ’ Dense(64) вҶ’ Dense(32) вҶ’ Dense(1)`
   optimized with Adam, MSE loss, and early stopping on validation loss.
7. Evaluate with RВІ, MAE, MSE, and RMSE.
8. Save the model and preprocessing artifacts for reuse in the app.

## рҹҡҖ App Features

- **Predict Price** вҖ” interactive form built dynamically from the saved
  feature list/encoders/defaults, with prediction history you can export.
- **Data Exploration** вҖ” upload a CSV to reproduce the notebook's outlier,
  correlation, and price-relationship charts.
- **Batch Prediction** вҖ” upload a CSV of multiple properties and download
  predictions for all of them at once.
- **About** вҖ” pipeline documentation and current model feature order.
